## ğŸš€ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Purab2001/Flagship-FaceOff.git
cd Flagship-FaceOff
```

2. Install dependencies:
```bash

# or

```

3. Start the development server:
